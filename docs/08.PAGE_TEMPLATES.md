# 2. Landing Page Template

Purpose: A focused conversion page for ads, campaigns, or
product-specific promotions.

Recommended structure:

1.  Header (minimal navigation)
2.  Hero Section
3.  Problem / Need
4.  Solution / Offer
5.  Key Benefits
6.  Social Proof / Credibility
7.  FAQ
8.  Strong CTA
9.  Footer

Template goal: Reduce distraction and drive action.

Rules: - fewer navigation options - strong CTA repetition - concise
copy - strong visual hierarchy

# 3. Service / Product Page Template

Purpose: Explain a service or product clearly and support search intent.

Recommended structure:

1.  Header
2.  Page Hero / Intro
3.  Product / Service Overview
4.  Core Benefits
5.  Detailed Features
6.  Comparison / Why Choose Us
7.  FAQ
8.  CTA
9.  Footer

SEO rules: - target intent-based keywords - use structured H2/H3
content - include rich descriptive copy

Recommended components: - feature cards - comparison table - CTA block -
FAQ details

Recommended structure:

1.  Header
2.  Brand Introduction
3.  Mission / Philosophy
4.  Core Values
5.  Team / Expertise
6.  Trust / Certifications / Numbers
7.  CTA
8.  Footer

# 5. Contact Page Template

Purpose: Help visitors contact the business quickly and confidently.

Recommended structure:

1.  Header
2.  Page Intro
3.  Contact Methods
4.  Contact Form or Inquiry CTA
5.  FAQ (optional)
6.  Location / Business Info
7.  Footer

Rules: - clear CTA - low friction - trust signals near form - semantic
form markup

------------------------------------------------------------------------

# 6. FAQ Page Template

Purpose: Support SEO and reduce user uncertainty.

Recommended structure:

1.  Header
2.  Page Intro
3.  FAQ Categories
4.  FAQ Items using
    ```{=html}
    <details>
    ```
    and
    ```{=html}
    <summary>
    ```
5.  Additional CTA
6.  Footer

Advantages: - search-friendly content - accessible interaction - easy to
maintain

Rules: - group questions logically - use keyword-aware questions - keep
answers concise but useful

------------------------------------------------------------------------

# 7. Blog / Insight Page Template

Purpose: Support informational SEO and content marketing.

Recommended structure:

1.  Header
2.  Article Hero
3.  Main Article Content
4.  Related Articles
5.  CTA / Newsletter / Contact
6.  Footer

Article rules: - one H1 - use H2/H3 properly - include internal links -
optimize images and alt text

Optional: - author info - publish date - structured data (Article
schema)

------------------------------------------------------------------------

# 8. Policy Page Template

Purpose: Display legal or policy information in a clean readable format.

Recommended structure:

1.  Header
2.  Page Title
3.  Policy Sections
4.  Contact Information
5.  Footer

decoration

# 9. Dashboard Entry Page Template

Purpose: Provide a simple entry screen for authenticated users.

Note: This belongs to the application layer, not the public site.

Recommended structure:

1.  Dashboard Header
2.  Summary Cards
3.  Recent Activity
4.  Quick Actions
5.  Data Table / Overview

Rules: - use app-layer framework if needed - keep layout modular -
prioritize usability over marketing

------------------------------------------------------------------------

# 10. AI Usage Rules for Page Templates

AI must:

1.  choose the closest matching template before generating a page
2.  follow semantic structure
3.  reuse components from COMPONENT_LIBRARY.md
4.  follow DESIGN_SYSTEM.md
5.  preserve SEO structure from SEO_RULES.md

AI must NOT: - invent a completely new page structure without reason -
mix dashboard layout patterns into public landing pages - remove
required trust / CTA / FAQ sections when they are strategically
important
