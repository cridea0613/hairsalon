# SEO_RULES

Defines mandatory SEO rules for all pages.

## Page Metadata

Every page must include: - unique title - meta description - canonical
tag

Title length: under 60 characters Description length: under 155
characters

## Heading Structure

-   One H1 per page
-   H2 for main sections
-   H3 for subsections

## Structured Data

Use JSON-LD schema when applicable: - Organization - Product - Article

## Image SEO

-   descriptive alt attributes
-   optimized sizes
-   WebP format
-   lazy loading

## Technical SEO

-   sitemap.xml
-   robots.txt
-   Open Graph tags
