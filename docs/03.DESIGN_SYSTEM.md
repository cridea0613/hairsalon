# DESIGN_SYSTEM

Defines the visual and layout rules for the project.

## Typography

Primary font stack: Inter, Pretendard, system-ui, sans-serif

Line height: 1.6

Heading scale: H1 40px H2 32px H3 24px Body 16px

## Spacing System

4px 8px 16px 24px 32px 48px

## Border Radius

6px 12px 16px

## Shadows

Soft Medium Strong

## Color Tokens

Primary: #007bff Text: #1a1a1a Border: #eeeeee Background: #ffffff

## Layout Rules

- Use generous whitespace
- Avoid clutter
- Prefer grid-based layouts
- Maintain consistent card components
